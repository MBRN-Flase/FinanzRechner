Datenschutz & Impressum

Keine Datenspeicherung
Alle Berechnungen erfolgen vollständig lokal in deinem Browser (JavaScript). Es werden keine Daten an Server übertragen, keine Cookies gesetzt und kein Tracking durchgeführt.

Externe Dienste
Live-Kurse werden von der öffentlichen CoinGecko API (coingecko.com) abgefragt. Dabei wird deine IP-Adresse an CoinGecko übertragen. Google Fonts werden von fonts.googleapis.com geladen.

Hinweis zur Besteuerung
In Deutschland gilt auf Kapitalerträge die Abgeltungssteuer von 25% + Soli (≈ 26,375%). Der Sparerpauschbetrag beträgt 1.000 € (2024). Dieses Tool zeigt vereinfachte Schätzungen — individuelle Steuerberatung empfohlen.

Keine Finanzberatung
Dieses Tool dient ausschließlich der illustrativen Darstellung mathematischer Zusammenhänge. Es stellt keine Anlageberatung im Sinne des WpHG dar. Vergangene Performance ist keine Garantie für zukünftige Ergebnisse.
