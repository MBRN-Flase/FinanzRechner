Datenschutz & Impressum

Keine Datenspeicherung
Alle Berechnungen erfolgen lokal in deinem Browser. Es werden keine Daten gespeichert oder übermittelt.

Externe API
Für Live-Kurse wird die öffentliche CoinGecko-API abgefragt. Keine Authentifizierung, keine personenbezogenen Daten.

Steuerlicher Hinweis
In Deutschland fällt auf Kapitalerträge die Abgeltungssteuer von 25% zzgl. Solidaritätszuschlag an. Das Tool zeigt Brutto-Werte. Steuerliche Beratung durch einen Fachmann empfohlen.

Keine Finanzberatung
Dieses Tool dient ausschließlich der Veranschaulichung. Es stellt keine Anlageberatung dar. Vergangene Performance ist keine Garantie für zukünftige Ergebnisse.
